VLoopCheck

Lightweight Error Checkers for protecting Vector Loops

Download this project as a .zip file Download this project as a tar.gz file

Introduction

VLoopCheck perform compiler-level error detector synthesis to protect vector loops by mining the compiler-level invariants by analyzing the code-generation patterns. Our work is based on the publicly available, state-of-the-art ISPC compiler.

Publication

Please refer to our paper published at DPDNS'16 for further details.
